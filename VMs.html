Setting Up Virtual Machines

Downloading ISOs

These tutorials will also be using Ubuntu Server ISOs. Like with the hypervisors, other distros can be used, but adapting these instructions for other distros is up to you. You can get the Ubuntu Server 20.04 ISO here. Another popular server distro is Fedora. You can get Fedora 35 here. It is recommended to keep your ISOs is their own folder wherever you want them. I keep mine on the desktop for quick access.

Installing VirtualBox

The proceeding tutorials will be using VirtualBox. Instructions will assume you are within that environment. The core of the instructions will work on other hypervizors or even on bare metal, but there may be differences in how it is set up. You can find the installer for VirtualBox here.
